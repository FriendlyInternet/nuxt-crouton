What changes

  • New danger-zone row. A Delete all orders row is added to the Event details card, directly under Delete event. Rendered for team admins only.
  • Typed-confirm modal, not a pill. Clicking opens a UModal where the admin must type the event name to enable the destructive button — a stronger guard than the Delete-event pill because it's bulk-destructive.
  • Scope is spelled out. The modal states it wipes only this event's orders + items + print jobs, keeps clients, and can't be undone.
  • Empties the list on success. After the delete lands, the event's order list refreshes to empty and a success toast fires.
  • Nothing else moves. Duplicate, Delete event, and every other Settings card stay exactly as they are.
