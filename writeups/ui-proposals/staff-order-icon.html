Staff-order switch: text label → icon

On a narrow (phone) pane the Dutch label “Personeelsbestelling” beside the staff-order switch runs off the edge of the cart's one-line controls row. Replacing the visible text with a compact person/badge icon (translated string kept as aria-label + tooltip) fits cleanly at any pane width.

What changes (and what doesn't)

  • Text label → icon. The visible {{ t('sales.cart.staffOrder') }} text is replaced by a person/badge glyph (i-lucide-user-cog) so the controls row never overflows on narrow panes.
  • Meaning stays reachable. The translated string is preserved as the icon's aria-label and a hover/tap tooltip — screen readers still announce “Personeelsbestelling / Staff order”.
  • Active state. When staff order is on, the icon adopts the warning tint (matching today's text-warning label emphasis) so the toggled state is still visible at a glance.
  • Scope. Only the staff-order control in Client/Cart.vue's footer changes. The remark toggle, the switch behaviour (update:isPersonnel), and everything else are untouched.
